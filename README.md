# Digital Complaint & Grievance System

A structured web-based platform for complaint submission, management, and real-time grievance tracking.

---

## Description

The Digital Complaint & Grievance System is a full-stack application designed to digitize and streamline the complaint handling process. It enables users to submit grievances through an online interface while providing administrators with tools to manage, update, and resolve complaints efficiently.

The system follows a layered architecture separating presentation, business logic, and data storage to ensure maintainability and scalability.

---

## Tech Stack

* **Frontend:** React
* **Backend:** Spring Boot
* **Database:** MySQL
* **API Architecture:** RESTful Services
* **Version Control:** Git & GitHub

---

## Architecture

Client–Server architecture with REST-based communication between frontend and backend services.

```
React (Client)
      ↓
Spring Boot (REST APIs)
      ↓
MySQL (Database)
```

---

## Core Modules

### User Module

* Authentication
* Complaint submission
* Complaint tracking
* Complaint history view

### Admin Module

* Authentication
* Complaint management
* Status updates
* Resolution handling
